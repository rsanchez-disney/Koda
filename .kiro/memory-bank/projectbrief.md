# Project Brief: Koda

## What is Koda?
Interactive terminal companion for steer-runtime. Manage agent profiles, chat with AI agents, orchestrate agent teams, and configure your environment — all from the terminal.

## Ecosystem
- **Kiro CLI** (`kiro-cli`) — Agent runtime engine
- **Kite** — Desktop GUI (Tauri + React)
- **Koda** — Terminal companion (Go + Bubbletea)
- **Steery** — Slack support bot
